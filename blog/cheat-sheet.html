Ruby Enumerables

About enumerables

Enumerables are mixins that provide collection classes with traversal and searching methods which have the ability to sort. Some common enumerables used are "each", "each_with_index", "map", "select", "inject",

each

This is a methods that belongs to Array and other collection classes. Each has the ability to iterate through a code block by each element in a collection. Below is an example.

sports = ['football', 'soccer', 'basketball', 'hockey']
sports.each do |x|
puts x
end

output: 'football' 'soccer' 'basketball' 'hockey'
Above, the each method allowed us to iterate through the array (sports) and output each element to the console. An important aspect of the each method is that it returns the same array unchanged. If you want to change an array, you need to use a different method.

each_with_index

Similar to the each method, each_with_index allows us to referrence the indices of an array. For example, if we wanted output only the even indices of an array, we would use the each_with_index method.
numbers = [1, 2, 3, 4, 5, 6]
numbers.each_with_index do |num, idx|
puts numbers if idx % 2 == 0
end
ouput: 2, 4, 6
As you can see above, we used each_with_index to interate through numbers to return only the even indices.

map

Map is a very similar method to each. It iterates through each method of an array and will return the elements by the code back. The major difference between each and map is that map will return a modified array modified by the code block. Let's show an example.
array = [1, 2, 3, 4, 5]
modified_array = array.map do |x| x * 2
end
puts array.join(", ") output: 1, 2, 3, 4, 5
puts modified_array,join(", ") output: -1, -2, -3, -4, -5
So what happened? Map returned a modified copy of the array.

select

Select is a method that takes in one argument. This block should pass some type of true and false method.
array = [1, 2, 'dog', false]
array.select {|x| x.class==Fixum}.join(', ')
output: 1, 2
As shown above, our program only selected elements of the array that were fixum.
